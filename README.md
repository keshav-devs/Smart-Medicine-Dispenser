# Smart Medical Dispenser

An Arduino-based smart medicine dispenser that automatically dispenses medication at scheduled times and sends SMS alerts using a GSM module.

## Features
- Automated pill dispensing
- LCD display interface
- GSM SMS alert system
- Servo motor pill rotation
- User acknowledgment system

## Technologies Used
- Arduino UNO
- Servo Motor
- GSM Module (SIM800L)
- LCD Display
- Embedded C (Arduino)
